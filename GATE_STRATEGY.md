# Full-Image Gate Strategy (v2)

## 선택: conf=0.20, r=40px (균형형)

### 방법
1. 풀이미지 640px 추론 (conf=0.20) → 대략적 사람 위치 후보
2. SAHI 타일 추론 (conf=0.50, 640x640, overlap=0.2, NMS/0.4/IOS) → 정밀 탐지
3. SAHI 결과 중 풀이미지 후보 중심 40px 반경 내만 채택

### 성능 (go2k_manual 604장 기준)

| 설정 | Precision | Recall | F1 | FP | FP감소 |
|---|---|---|---|---|---|
| SAHI only (baseline) | 0.725 | 0.906 | 0.804 | 575 | - |
| **gate conf=0.20, r=40px** | **0.867** | **0.893** | **0.880** | **231** | **-344 (60%)** |

### 파라미터
- `GATE_CONF = 0.20` - 풀이미지 최소 confidence
- `GATE_RADIUS = 40` - 풀이미지 탐지 중심에서 반경 (px)
- `SAHI_CONF = 0.50` - SAHI 최종 confidence

### 적용 대상
- pseudo-label 생성 파이프라인 (적용 완료)
- 실시간 추론에는 게이트 미적용 (SAHI only) — 모델 품질 향상으로 FP 감소 추구

### 적용 결과 (captures 3K)
- 게이트 전 bbox: 6,233개
- 게이트 후 bbox: 5,058개 (1,175개 FP 필터, 19%)
